Class: Character

Character()

Represents a character (player) in the game. The character can perform actions like walking, jumping, and throwing bottles, as well as managing health, loot, and animations.

Constructor

new Character()

Creates an instance of the Character class. Initializes the character with idle, walking, jumping, hurt, and death animations.
