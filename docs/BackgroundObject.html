Class: BackgroundObject

BackgroundObject(imagePath, x)

Represents a background object in the game world. This class is used to create and manage the background elements that move along with the game. The background object can be an image that moves horizontally based on the game state.

new BackgroundObject(imagePath, x)

Creates an instance of the `BackgroundObject` class.
Parameters:
Name Type Description
